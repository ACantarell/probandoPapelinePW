# Para isntalar PW se puede hacer de disntitnas formas
-por consola poniendo npx init playwright
-aprentando CRT+SHIFT+P y poner "instal playwright"

# Como para emular a un usuario se puede hacer de la siguiente formas, en la consola poner:

-npx playwright codegen 
y esto lo que hace es abrite 2 ventanas. una de ellas es para emular un usuario y la otra escribe codigo de lo que hace la usiaria.

## Comandos para usar test y reportes son:

  npx playwright test
    Runs the end-to-end tests.

  npx playwright test --ui
    Starts the interactive UI mode.

  npx playwright test --project=chromium
    Runs the tests only on Desktop Chrome.

  npx playwright test example
    Runs the tests in a specific file.

  npx playwright test --debug
    Runs the tests in debug mode.

  npx playwright codegen
    Auto generate tests with Codegen.

We suggest that you begin by typing:
    npx playwright test
